# Security Policy

If you discover a security vulnerability in these materials, please do not open a public issue. Instead, contact the repository maintainers via email: security@octoacme.example (replace with the appropriate contact for your organization), or use GitHub's private security advisories feature.

Please include:
- A short description of the issue
- Steps to reproduce
- Any proof-of-concept or exploit code (if available)

Maintainers will acknowledge receipt within 72 hours and coordinate remediation and disclosure steps.
