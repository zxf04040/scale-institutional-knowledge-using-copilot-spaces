# Contributor Covenant Code of Conduct (summary)

We expect contributors to follow a respectful, inclusive, and professional code of conduct. This repository adopts the Contributor Covenant as its code of conduct; please read the full text at https://www.contributor-covenant.org/version/2/1/code_of_conduct/

If you experience or witness unacceptable behavior, please report it to the maintainers by opening an issue or contacting the project owners directly.

Maintainers will respond to reports confidentially and take appropriate action.
